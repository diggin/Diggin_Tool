Diggin_Tool
===========
- with ZendFramework-1.10

Quick to have YOUR "zf command"

### install ###
- set include_path
- copy 'bin/' folder's diggin & diggin.php to YOURS-bin and YOURS.php
- edit YOURS-bin's "diggin.php"
